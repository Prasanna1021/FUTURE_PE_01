# AI Website Copy Generator for Local Businesses

## Future Interns – Prompt Engineering Task 1 (2026)

### Project Overview

This project demonstrates how Prompt Engineering and AI can be used to generate conversion-focused website copy for local businesses.

For this task, I selected a local beauty salon and created a complete website content system, including homepage copy, service descriptions, call-to-action sections, and a website prototype using AI-generated content.

The goal was to create professional, publish-ready website content that improves:

* Clarity
* Customer Trust
* Brand Positioning
* Conversion Rates
* User Experience

# Objective

To design a reusable prompt framework capable of generating:

* Homepage Website Copy
* Service Page Content
* Call-to-Action (CTA) Sections
* Tone-Adaptive Business Content

The generated content was then implemented into a website prototype.
